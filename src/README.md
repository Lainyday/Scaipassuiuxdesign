# SC AI-Pass - Internal AI Platform

A complete UI/UX design for SC AI-Pass featuring a Toss-inspired aesthetic with a clean, minimalist, and friendly super app feel.

## Design System

### Colors
- **Primary:** SK Orange `#F47920`
- **Background:** Light Gray `#F2F4F6`
- **Card:** Pure White `#FFFFFF`
- **Text Primary:** High Contrast Black `#191F28`
- **Text Secondary:** Soft Gray `#8B95A1`

### Design Principles
- **Minimalist & Clean:** All content in white cards with 24px rounded corners
- **Bold Typography:** Heavy weights for numbers and main titles
- **Friendly 3D Icons:** Colorful, soft, gradient-based icons
- **Deep Rounded Corners:** 24px radius for cards, 16-20px for smaller elements
- **Subtle Shadows:** Soft drop shadows for depth

## Features

### Mobile App (iPhone View)
1. **Login & Onboarding** - Large 3D character with SK Orange CTA button
2. **AI Chat (Home Tab)** - Empty state with suggestion pills, chat bubbles with rounded corners
3. **My Status (Dashboard)** - 3D level badge, progress bars, action cards
4. **Today's Mission** - Daily quest cards with colorful 3D icons
5. **Community Feed** - Card-based prompt sharing with "Try this Prompt" buttons
6. **L2 Application Form** - Clean single-column form with calculated fields

### Desktop Web (1920x1080)
1. **Vertical Sidebar** - Navigation menu with logo and user profile
2. **Main Chat Area** - 60% width center area for AI interactions
3. **Right Helper Panel** - 20% width showing status summary and missions
4. **Community Grid** - Pinterest-style 3-column grid layout

### Admin Dashboard
- **Metric Cards** - Large numbers with colorful gradient icons
- **Application Management** - Clean table with approve/reject actions
- **Recent Activity** - Timeline of user actions

## Navigation

Use the view switcher in the top-right to toggle between:
- **Mobile** - iPhone-optimized mobile experience
- **Desktop** - Full HD desktop layout (1920x1080)
- **Admin** - Administrator dashboard

## Key Components

- Level badges with 3D medal effects
- Gradient backgrounds for feature highlights
- Rounded pill buttons for quick actions
- Progress bars with gradient fills
- Avatar circles with gradient backgrounds
- Card-based layouts throughout
- Tab navigation for mobile

## Technology Stack

- React with TypeScript
- Tailwind CSS v4.0
- Lucide React icons
- ShadcN UI components
